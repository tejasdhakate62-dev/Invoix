# Invoix

A sleek, intuitive mobile application designed for freelancers and small businesses to create, track, and manage invoices seamlessly. With a modern dark-themed UI and a streamlined navigation architecture, **Invoix** simplifies your billing workflow so you can focus on growing your business.

**Tagline:** CREATE. SEND. GET PAID.

## ✨ Features

*   📊 **Financial Metrics Dashboard:** Monitor your incoming cash flow efficiently with clear summary cards detailing your active earnings and outstanding balances.
*   🔍 **Smart Transaction Filtering:** Instantly sort and track documents using localized filter pill selectors categorized by *All Invoices*, *Paid Only*, and *Pending Only*.
*   📝 **Robust Billing Creator:** Append multiple line items, evaluate custom tax rate percentages, and finalize templates instantly via a structured data entry portal.
*   🌐 **Localization Support:** Configure your preferred native monetary symbols globally via the localization parameters menu.
*   🏢 **Comprehensive Profiles:** Keep generated documents legally compliant by saving standard company identifiers alongside high-resolution business branding logos.
*   🔐 **Secure Access Control:** Protect sensitive user data with robust authentication components, fallback reset routing, and OAuth compatibility.
*   🎨 **Tailored Appearance Modality:** Built to interactively synchronize with native device display conditions, supporting System Default configurations alongside manual Dark/Light overrides.

## 🛠️ Tech Stack
